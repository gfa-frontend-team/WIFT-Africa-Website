# Analytics API

> [!WARNING]
> **BETA FEATURE**: This module is currently in **Beta**. Endpoints and response structures are subject to change.
> Routes may not be active in all environments.

The Analytics module provides data insights for Users (Post performance) and Admins (Connection health).

## Base URL
`/api/v1/analytics`

## Key Features
- **Post Analytics**: Track impressions, reach, engagement, and audience demographics for posts.
- **Connection Analytics**: (Super Admin) Monitor platform-wide networking health and inter-chapter connections.

## Sub-Modules
- **Posts**: `/analytics/posts`
- **Connections**: `/analytics/connections`
