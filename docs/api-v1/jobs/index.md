# Job Board API

The Job Board module allows administrators to post job opportunities and users to view and apply for them.

## Base URL
`/api/v1/jobs`

## Key Features
- **Job Listing**: Publicly viewable job board with filtering.
- **Job Management**: Admin tools to create, update, and archive jobs.
- **Application**: Users can apply directly to internal jobs or be redirected for external ones.
