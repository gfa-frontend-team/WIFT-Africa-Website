# Job Applications API

The Job Application module manages user applications to job postings.

## Base URL
`/api/v1/application`

## Key Features
- **My Applications**: Users can track their application history.
- **Application Details**: View status, cover letter, and resume of submitted applications.
- **Admin Management**: Admins can review applications and update their status (Shortlisted, Rejected, Hired).

## Related Modules
- [Jobs](../jobs/index.md) - The jobs being applied to.
